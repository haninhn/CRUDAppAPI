# CRUDAppAPI
The project includes comments to aid anyone looking to learn Node.js with Mango DB. Node JS CRUD app (login, add, delete, update).
Fist of all 
1- Installation: 
    Download mongoDB
    download postMain 
    download node js
    
2- Install package.json
    npm init 
    
3- Run server                                             
    node server.js
    
4- Install express 
   npm Install express
